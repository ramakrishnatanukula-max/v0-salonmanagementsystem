## Responsive Breakpoints

### Mobile (< 640px)
```
- Full width cards
- Stack buttons vertically
- Reduce padding
- Responsive fonts
- Hide unnecessary columns
```

### Tablet (640px - 1024px)
```
- Grid columns: 2
- Better spacing
- Horizontal button layout
- All information visible
```

### Desktop (> 1024px)
```
- Grid columns: 3+
- Maximum width: 1152px (6xl)
- Full spacing
- All features visible
```

## Transitions & Animations

### Hover Effects
```
Buttons:        scale(1.05) + shadow increase
Cards:          bg-gradient + shadow increase
Inputs:         ring-2 ring-color on focus
Checkboxes:     accent-color highlight
```

### Auto Animations
```
Toast:          fade-in + slide-in-from-top
Modal:          backdrop blur + smooth scale
Loading:        spin animation on button
```

## Accessibility

### Color Contrast
```
Text on Background:    WCAG AA compliant (4.5:1+)
Buttons:               Sufficient contrast
Links:                 Underlined or bold
```

### Keyboard Navigation
```
Tab Order:             Logical flow
Focus Indicators:      ring-2 visible
Form Labels:           Associated with inputs
```

### Screen Readers
```
Semantic HTML:         Proper heading hierarchy
ARIA Attributes:       role="alert" for toasts
Button Labels:         Descriptive text
```
